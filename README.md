# ✈️ Flight Radar App

This project is a fully responsive flight radar application built with modern web technologies. Users can track live flights on an interactive map, view detailed flight information, and browse all available flights in a list.

# 🚀 Technologies Used
- React
- react-router-dom
- @reduxjs/toolkit
- react-redux
- @splidejs/react-splide
- leaflet
- react-leaflet
- react-paginate
- bootstrap
- axios
# 📌 Features
- 🌍 Live Flight Tracking: Displays real-time flights on the map.
- 🔢 Instant Flight Count: Shows the number of active flights on the screen.
- 🗺️ Map Integration: Flights are visualized on an interactive map.
- ✈️ Flight Detail Page:
- Flight route
- Aircraft specifications
- Detailed flight information
- 🖱️ Interactive Experience: Click on aircraft icons to navigate to the detail page.
- 📋 Flight List Page: Displays all flights in a list view.
- 🔄 Navigation: Users can select a flight from the list and go to its detail page.
- 📱 Responsive Design: Optimized for all devices (mobile, tablet, desktop).
